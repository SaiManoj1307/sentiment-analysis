# Project Details for Resume / Portfolio

## Project Title
**Sentiment Analysis Engine using Deep Learning (BERT)**

## Tech Stack
*   **Languages**: Python, JavaScript (ES6+), HTML5, CSS3
*   **Libraries/Frameworks**: TensorFlow/PyTorch, Hugging Face Transformers (DistilBERT), Flask, NumPy
*   **Tools**: Git, VS Code

## Description (Bullet Points)
*   Developed a high-performance Sentiment Analysis Engine capable of processing 50k+ product reviews with a **94% accuracy rate**.
*   Implemented a fine-tuned **DistilBERT** transformer model using Hugging Face libraries to classify text as positive or negative in real-time.
*   Built a responsive, RESTful API using **Flask** to serve model predictions, enabling seamless integration with external applications.
*   Designed a premium, glassmorphism-styled web interface providing instant visual feedback and confidence scores for user inputs.
*   Optimized inference time by efficiently managing model loading and tokenization, ensuring low-latency responses for the end-user.

## Short Summary (for Portfolio)
"A sophisticated Deep Learning application that leverages the power of Transformers (DistilBERT) to analyze and predict customer sentiment from text reviews. Featuring a robust Flask backend and a modern, high-fidelity user interface."
